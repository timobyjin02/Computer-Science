## Why
cs지식이란 컴퓨터 공학적 지식을 말한다.
디자인패턴, 데이터 베이스 운영체제, 네트워크등을 포함한다. 이런 지식은 개발자가 비지니스 로직을 구축하는 데 있어서 기초가 되는 상식이고 기술 면접에 대비하기 위해서 공부해야 한다.
